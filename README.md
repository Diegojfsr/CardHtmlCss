# CardHtmlCss
Criação de cards com HTML e CSS


### O Projeto
A estrutrura do projeto você pode pegar aqui no repositório (as imagens estão na pasta assets), tente criar você mesmo, apenas atenção a alguns pontos:  
> 1. Lembre de referenciar o arquivo style.css no arquivo index.html
> 2. Crie as referências das imagens também
> 3. Defina a classe card no seu container

   
O seu projeto já está com o arquivo de estilo linkado basta experimentar.
> background-color: define uma cor de fundo do container
> box-shadow: define sombra para os cards
> font-size: altera o tamanho da fonte
